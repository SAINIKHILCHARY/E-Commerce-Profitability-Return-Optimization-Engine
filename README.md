# E-Commerce Profitability & Return Optimization Engine

## Project Overview

This project focuses on analyzing e-commerce transaction data to identify opportunities for improving profitability and reducing product returns. Using SQL, Python, and Power BI, the project will generate business insights related to sales performance, customer behavior, product profitability, and return optimization.

## Dataset

**Source:** Kaggle – E-commerce Business Transaction Dataset

The dataset contains **536,350 transaction records** and **8 columns**.

### Dataset Schema

| Column        | Description                   |
| ------------- | ----------------------------- |
| TransactionNo | Unique transaction identifier |
| Date          | Date of the transaction       |
| ProductNo     | Unique product identifier     |
| ProductName   | Name of the product           |
| Price         | Unit price of the product     |
| Quantity      | Quantity purchased            |
| CustomerNo    | Unique customer identifier    |
| Country       | Customer's country            |

## Project Objectives

* Analyze sales and revenue trends.
* Identify high-performing and low-performing products.
* Measure customer purchasing behavior.
* Detect transactions that represent returns or cancellations.
* Build profitability KPIs and dashboards.
* Recommend strategies to reduce returns and improve profit margins.
